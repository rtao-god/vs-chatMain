## Getting Started

Add ChatWindow lib using symlink:
```bash
ln -s /path/to/repo/vs_chat_widget_front/src/ChatWidndow ./components/ChatWindow
```

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
